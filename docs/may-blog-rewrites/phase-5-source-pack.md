## Official sources — Oakland release

- [Towed Vehicles | City of Oakland](https://www.oaklandca.gov/Public-Safety-Streets/Parking-Vehicles/Towed-Vehicles)
- **Auto Plus Towing:** (510) 261-8199; public plate/VIN lookup on website
  - Hours: Mon–Fri 8 AM–5 PM; weekends/holidays 8 AM–3:30 PM
- **OPD Records:** (510) 238-3021 (24/7 status)
- **OPD Records Division:** 455 7th Street, Room 306, Oakland CA 94607
  - Hours: Mon–Fri 8 AM–3 PM; Sat/Sun/holidays 8 AM–noon
- Administrative release fee: $166 standard impound (city tow resource guide—verify current)
- STOP impound release fee: $280 (city tow resource guide)
- Tow triggers: stolen recovered, boot unpaid, reg expired 6+ months, $2,500+ Oakland tickets, 5+ CA tickets, 3-day abandon tag
- [Abandoned Auto Unit | OakDOT](https://www.oaklandca.gov/Government/Departments/Transportation-OakDOT/Abandoned-Auto-Unit)

## Official sources — Oakland hearings

- **Administrative Hearing Examiner:** (510) 238-4484 — tow hearing requests and fee waiver if towed in error
- Only registered/legal owner or agent may dispute tow and fees
- CVC §22659.5 — poststorage hearing notice and 10-day request window (state law)
- CVC §14602.6 — 30-day impound hearing for suspended/revoked/unlicensed driver
- Evidence holds require investigator/court release before Records
